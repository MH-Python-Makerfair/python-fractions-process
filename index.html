Okay. So the idea I had, was to ask the user to enter numbers, and they would receive more numbers, right? That was just the basic outline. But I did like the whole interactive-user-thing, so I decided that I wanted to ask the user the denominator/numirator of the first and second fraction, and then I would tell them the answer to the problem. And I knew I could do this with the input() python function:
numone = input()

displayed here, I have the variable numone, which stands for numerator one (the numirator of the first fraction). What it's saying is that whatever number the user types, will be stored in the variable numone, so whenever I want to display numone, it will output the users input. It might sound confusing, but it can help to think of it like algebra variables. Now we have but two problems. One, cleaning up the code, and two, numbers (or integers, which is the proper name in python). See, if I wanted to multiply a variable by a variable, python says no. Basically, you have to define hat the variable is equal to a number. Because you can't multiply words, obviously. how an I fix this? The int() function (it tells python that this variable is indeed, an integer.) Here's what the code would look like:
numone = int(input("Enter the first fraction's numerator:"))

We just took the input() function, and put it inside the int() function. And we also moved the prompt or "enter the (blah blah blah)" inside of the input() function's parentheses, because that fixes problem one, making the code look better, and it also happens to help with errors.
